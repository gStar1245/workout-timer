# 💪 Workout Timer

덤벨 홈트용 운동 타이머 웹앱

## 기능

**타이머**
- 세트 완료 탭 (또는 Enter) → 자동 휴식 카운트다운
- 휴식 시간 조절 (20~90초 / 5초 단위)
- 세트 수 선택 (1~5)
- 종목 완료 시 자동 다음 종목 이동
- 완료음 / 진동 알림
- 화면 자동 꺼짐 방지 (Wake Lock)
- 운동 시작 시각 표시

**설정**
- 템플릿 5종 (전신 덤벨 / 상체 덤벨 / 하체 덤벨 / 어깨·팔 / 맨몸 전신)
- 종목 추가/삭제/초기화 (최대 10개)
- 소리 ON/OFF
- 다크 / 라이트 테마

**기록**
- 최근 운동 3개 빠른 선택 (localStorage)
- 운동 기록 JSON 복사 (Notion 연동용)
