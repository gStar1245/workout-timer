# 운동 타이머 (GymStack V1)

홈 덤벨 운동을 위한 모바일 웹 타이머.
단일 HTML 파일, 외부 의존성 없음.

🔗 https://gstar1245.github.io/workout-timer

---

## 기능

**타이머**
- 세트 완료 탭(또는 Enter) → 자동 휴식 카운트다운
- 종목 완료 후 휴식: "종목 완료 · 탭하면 스킵" 표시 (세트 간 휴식과 구분)
- 휴식 시간 20~90초 / 5초 단위 조절
- 세트 수 1~5 선택
- Wake Lock 화면 꺼짐 방지
- 완료음 / 진동 알림 (소리 ON/OFF)

**종목 / 템플릿**
- 템플릿 5종: 전신 덤벨 / 상체 덤벨 / 하체 덤벨 / 어깨·팔 / 맨몸 전신
- 타이머 탭 "종목 목록" 옆 ✏ 편집 버튼 → 바텀시트
  - 템플릿 선택 → 즉시 적용 (운동 재시작)
  - 종목 순서 변경 / 추가 / 삭제
  - 적용 시 세션 유지, 기존 세트 기록 보존
- YouTube 검색 연동 (종목명 옆 버튼 → 새 탭)
- 최근 운동 3개 칩 UI (localStorage)

**기록**
- 운동 완료 이력 JSON 복사 (Notion 연동용)
- 완료 시 localStorage 자동 저장

**UI**
- 다크 / 라이트 테마
- 응원 메시지 랜덤 출력
